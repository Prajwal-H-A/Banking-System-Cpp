Banking System in C++

A simple Banking Management System built using C++, focusing on Object-Oriented Programming (OOP), STL containers, and File Handling concepts.

Features

Create new bank accounts

Deposit and withdraw money

Close accounts

Check balance and view all accounts

Persistent data storage using files (account_list.txt, closed.txt, account_number.txt)

Concepts Used

C++ OOP (Classes, Objects, Friend Functions)

STL Containers: map, set

File I/O: ifstream, ofstream

Operator Overloading

Static Members and Encapsulation
